# RestoFlow - Landing Page 🚀

Bienvenido al repositorio de la Landing Page oficial de **RestoFlow**, tu Sistema POS Híbrido premium para restaurantes, franquicias y agencias de desarrollo.

## 🌟 Características Principales

*   **Rendimiento Ultrarrápido:** Construida con Astro Framework para entregar una experiencia fluida sin tiempos de carga innecesarios.
*   **Diseño Premium (UI/UX B2B):** Interfaz basada en *glassmorphism*, fondos dinámicos y micro-animaciones diseñadas para captar leads "High-Ticket".
*   **Responsive Design:** Adaptación impecable a cualquier dispositivo (Móvil, Tablet o Desktop) sin generar sobrecargas visuales.
*   **Embudo de WhatsApp Integrado:** El formulario de demostración no recoge datos al aire; estructura la información (Nombre, Email, Empresa y Rol) y la envía pre-formateada al WhatsApp de soporte (+59179717725) para agendar de inmediato.

## ⚙️ Stack Tecnológico

*   **Framework:** [Astro](https://astro.build/)
*   **Estilos:** CSS3 Nativo Avanzado (Variables de Color, Animaciones Clave, Clamp Typography).
*   **Lógica Frontend:** TypeScript / JavaScript Vanilla.

## 🚀 Despliegue Local

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/Ggus23/Landing-RestoFlow.git
    cd Landing-RestoFlow
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    # o si usas bun/yarn:
    bun install / yarn install
    ```

3.  Levanta el servidor de desarrollo:
    ```bash
    npm run dev
    ```

Abre tu navegador y explora el ecosistema visual de RestoFlow.

## 📂 Estructura de Directorios

- `src/layouts/` — Contenedor principal (`Layout.astro`) donde radican los Meta Tags y la inyección global.
- `src/components/` — Los "ladrillos" de la UI (Navbar, Hero, Problema vs Solución, Contacto, etc).
- `src/styles/` — Archivos de cascada globales como `global.css` que dictan el *Design System*.
- `public/` — Assets públicos servidos estáticamente (ítems como Favicon).
